ℹ️ Fonctionnement

La clé API OpenAI est configurée côté serveur, dans le fichier .env du backend Trekko.

Elle n'est jamais stockée dans le navigateur.

Pour configurer : ajoutez OPENAI_API_KEY=sk-... dans .env et lancez le backend (cd server && npm start).

📘 Instructions d'installation
  1. Créez une clé sur platform.openai.com
  2. Copiez .env.example → .env
  3. Ajoutez votre clé dans .env
  4. Lancez : cd server && npm install && npm start
  5. Cliquez "Tester la connexion" ici

⚠️ L'API OpenAI est facturée séparément de ChatGPT Plus. Chaque circuit génère ~0.01-0.05€ de coût API.
